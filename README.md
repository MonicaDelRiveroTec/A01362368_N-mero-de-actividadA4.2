# Project Results

This repository contains three programs, each organized in its own folder with source code and results.

## Structure

Each program folder contains:
- **source/** - The program's source code
- **results/** - Output and analysis results

## Results Documentation

The PDF file in this repository includes:
- Results from each program run
- Pylint analysis for all three programs
